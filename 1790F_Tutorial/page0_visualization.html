Page 0: The Problem - Timofey and Black-White Tree

Before diving into the code, let's understand the problem visually. We are given a tree where initially, one vertex is painted black and all others are white.

In each step, one new vertex is painted black. We need to output the minimum distance between any two black vertices after each step.

Current Step: 0 | Min Distance: ∞

Notice how the minimum distance only stays the same or decreases. It never increases! This non-increasing property is the key to solving the problem efficiently, leading us to our BFS pruning strategy.
