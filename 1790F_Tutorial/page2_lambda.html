Page 2: The Lambda Function & BFS

What does the Lambda function mean?

auto update = [&](int start_node) { ... };

This line defines a lambda function in C++. A lambda is basically a quick way to write an anonymous, inline function inside another function (like inside src()).

The [&] part is the "capture clause". It tells the compiler to capture all local variables from the surrounding function by reference. This means update can directly read and modify min_dist_to_black, ans, and adj without passing them as arguments!

What does it actually do?

The update function performs a Breadth-First Search (BFS) starting from a newly colored black node.

Its primary job is to traverse the tree and update the min_dist_to_black array for other neighboring nodes, ensuring every node knows its shortest distance to *any* black node.
